# MusicFlow: Short-Form Music Videos

This project is a web-based application that brings the short-form video experience to music discovery. It's a prototype of a platform where users can browse and interact with a feed of short-form music videos.

## Features

*   **Vertical Video Feed**: A vertically scrollable feed of short-form music videos, providing an immersive and engaging user experience.
*   **Interactive UI**: Users can like, comment, and share videos, fostering community engagement.
*   **Music Discovery**: Each video is linked to a song and artist, allowing users to discover new music and artists.
*   **Artist and Album Pages**: Seamless navigation to dedicated pages for artists and albums.
*   **Genre-based Feeds**: Users can explore different genres and view curated video feeds for each.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key (Note: This project does not seem to use this key, but it is in the original README).
3. Run the app:
   `npm run dev`
